### How to Run Application
-  Configure enviroment variable .env

```
NODE_ENV=dev
DATABASE_URL=

```

- Run Application Dev
```
npm run start:dev
```
- Run Application Production
```
npm run start
```

- Run Test
```
npm run test
```
- Generate Build
```
npm run build
```

- Run code coverage
```
npm run test:coverage
```
